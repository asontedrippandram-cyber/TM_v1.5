# 车载软件测试进度看板 - SPEC.md v4.0

## 1. 界面设计原则

- **风险为核心**：顶部风险卡片一览无余
- **一键聚焦**：点击卡片直接展开详情
- **信息分层**：主次分明，视觉引导清晰
- **操作直观**：所见即所得，无需多步探索

## 2. 布局结构

```
┌──────────────────────────────────────────────────────────┐
│  Header: 标题 + 项目选择器                                 │
├──────────────────────────────────────────────────────────┤
│  风险看板（顶部横向，滚动）                                  │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐           │
│  │ 奥迪A4 │ │ 奥迪A6 │ │ 宝马3系│ │ 宝马5系│           │
│  │  🔴高  │ │  🟡中  │ │  🟢低  │ │  🟡中  │           │
│  └────────┘ └────────┘ └────────┘ └────────┘           │
├──────────────────────────────────────────────────────────┤
│  选中车型详情                                              │
│  ┌────────────────────────┬─────────────────────────────┐ │
│  │  五维度进度（60%）       │  测试用例表格（40%）         │ │

│  └────────────────────────┴─────────────────────────────┘ │

└──────────────────────────────────────────────────────────┘
```

## 3. 风险卡片设计

### 卡片尺寸
| 风险等级 | 尺寸 | 边框颜色 | 背景色 |
|----------|------|---------|--------|
| 高风险 | 180px × 120px | #ef4444 | rgba(239,68,68,0.08) |
| 中风险 | 150px × 100px | #f59e0b | rgba(245,158,11,0.08) |
| 低风险 | 120px × 80px | #10b981 | rgba(16,185,129,0.08) |

### 卡片内容
```
┌─────────────────────┐
│  奥迪 A4        🔴  │  ← 车型名 + 风险图标
│  距商用: 89天       │  ← 距离商用天数
│  ████████░░ 85%   │  ← 整体完成度
│  未完成: 3用例 2维度 │
└─────────────────────┘
```

## 4. 五维度进度（选中后显示）

保持原有五维度串联设计，增加：
- 每个维度可点击编辑
- 前2维度显示用例统计
- 后3维度纯手动

## 5. 测试用例表格

- 显示当前选中车型的用例
- 支持按状态、优先级筛选
- 新增用例默认关联当前选中车型

## 6. 侧边栏

- 默认收起，显示展开按钮
- 展开后显示特性树
- 点击特性可筛选用例

## 7. 交互流程

1. 打开页面 → 顶部显示所有车型风险卡片
2. 点击任意卡片 → 该车型高亮，卡片下方展开详情
3. 编辑五维度进度 → 点击编辑按钮 → 弹窗修改
4. 新增用例 → 自动关联当前选中车型

## 8. Acceptance Criteria

1. 打开页面，顶部横向显示所有车型风险卡片
2. 卡片大小/颜色对应风险等级
3. 点击卡片直接展开五维度详情
4. 侧边栏默认收起，不干扰主流程
5. 编辑进度弹窗简洁直观
6. 数据持久化正常
